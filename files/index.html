MoodleWatson

This is a project that uses a combination of IBM Cloud and Moodle to create a perpetual class with revolutionary prospects to both students and professors!

Perpetual Class Overview

The perpetual class will be an Earth and Space Science course. Within it, the student can ask questions to the chatbot such as "when is my next test?" or "what assignments are due next week?" and the interactive chatbot will provide the relevant answer.
